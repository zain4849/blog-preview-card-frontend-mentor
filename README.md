# Blog Preview Card

A responsive blog preview card component built with HTML and CSS.

## Features

- Hover animations on card and title
- Responsive design
- Clean, modern layout
- Custom CSS variables for colors and spacing

## Tech Stack

- HTML5
- CSS3
- Google Fonts (Figtree)
